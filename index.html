O que são Tags HTML?

(Voltar)

As tags HTML são elementos fundamentais na construção de páginas da web. Elas são usadas para marcar e estruturar o conteúdo, permitindo que os navegadores interpretem e exibam informações de maneira adequada. Elas são como etiquetas ou rótulos que envolvem o conteúdo em um documento HTML.

As tags delimitam diferentes partes do conteúdo, como títulos, parágrafos, links, imagens e listas. Os navegadores interpretam essas tags para renderizar a página corretamente.

Elas são escritas entre os sinais < e >. Existem tags que precisam de fechamento (como <.tag>...) e tags que não precisam (como <.tag />).

Exemplos de Tags:

  • <.h1><./h1>, <.h2><./h2>, <.h3><./h3>, <.h4><./h4>, <.h5><./h5>, <.h6><./h6>: Definem o tamanho do título do documento com diferentes níveis de importância.
  • <.p><./p>: Define o começo e o fim de um parágrafo. Pode ser utilizado ao lado de align para indicar o alinhamento do trecho.
  • <.a><./a>: Cria um link e atribui elementos a ele.
  • <.img/>: Permite inserir imagens na página web e não possui fechamento, contendo apenas atributos nela.
  • <.i><./i>, <.u><./u>: Formatam textos em itálico, e sublinhados, respectivamente. Ambas requerem abertura e fechamento com o conteúdo entre elas.
  • <.br><./br>: Utilizada para quebra de linha em um texto. É mais comum a utilização da tag fechada <./br> pois ela é suportada tanto em HTML quanto XHTML.
  • <.strong><./strong>: Utilizada para representar uma parte do texto que seja importante pois o texto fica em negrito no navegador web.
  • <.span><./span>: Utilizada para agrupar elementos para fins de estilo, porém só deve ser usado quando não houver outra alternativa de elemento semântico.
  • <.audio><./audio>: É utilizada para definir sons como música ou outros tipos de áudio. O HTML5 suporta mp3, onda e ogg.
  • <.video><./video>: É usada para importação de arquivos de vídeo. O HTML suporta três formatos de vídeos compatíveis com essa tag (mp4, webM e ogg).
  • <.ol><./ol>: Cria listas ordenadas. Deve ser acompanhado de start, para indicar o começo da lista e type, para apontar o tipo de caractere que será ordenado.
  • <.ul><./ul>: Cria listas não ordenadas. Precisa do type para apontar o tipo de item que fará parte da lista.
  • <.li><./li>: Define cada item da lista, que fazem parte de listas ordenadas ou não ordenadas.
  • <.style><./style>: Define o estilo de parte ou de todo o conteúdo da página web com arquivo .css importado.
  • <.script><./script>: Utilizada para a importação de um documento Javascript para interações do lado cliente com conteúdo dinâmico.
  • <.input type=”text”>, <.input type=”email”>, <.input type=”Submit”>: São utilizadas para o recebimento de dados do usuário, possuindo diferença entre elas como o tipo do atributo que irá receber (type).

Estrutura básica de um documento HTML

(Voltar)

A estrutura básica de um documento HTML consiste em 5 elementos: <.!DOCTYPE>, <.html><./html>, <.head><./head>, <.title>, <.body><./body>.

  • O <.!DOCTYPE> informa ao navegador da web em qual versão do HTML a página está sendo escrita, por exemplo quando incluímos dentro do DOCTYPE a palavra HTML (<.!DOCTYPE html>), o navegador entenderá que aquela versão é do HTML5.
  • O elemento <.html><./html> está logo abaixo da declaração DOCTYPE, é o elemento básico e principal de um documento HTML e atua como um container onde podemos definir o idioma padrão utilizado pelo documento através do atributo “lang” que por padrão vem em inglês (<.html lang = “en”>). Todo documento HTML inicializa e finaliza com essa tag.
  • O <.head><./head>é o cabeçalho do documento cujo conteúdo não é exibido no navegador durante o carregamento da página. Esse elemento contém metadados, informações importantes, que especifica dados sobre o documento HTML.
  • O elemento <.title>é usado para definir o título do documento que é exibido na parte superior da janela do navegador, e fornece um título para a página quando ela é adicionada aos favoritos, favorecendo a busca nos resultados de pesquisa.
  • O <.body><./body> define o corpo do documento, e tudo que estiver dentro desta tag será mostrado no navegador web.

O que são atributos HTML?

(Voltar)

Atributos HTML são informações adicionais que podem ser incluídas nas tags HTML para modificar o comportamento ou a aparência dos elementos na página. Eles são escritos como pares de nome e valor, dentro das tags. Alguns pontos importantes a serem destacados sobre atributos são: a sintaxe geral de um atributo é: nome="valor"; o nome do atributo é sempre escrito em letras minúsculas, o valor pode ser uma string, número ou outra informação relevante.

Exemplos de Atributos:

  • class: define uma classe CSS para estilização. Vários elementos podem compartilhar a mesma classe.
  • href: define a URL de destino para links.
  • target: identifica se o link deve ser aberto na mesma janela, em nova aba ou nova janela;
  • id: identifica exclusivamente um elemento. Deve ser único na página.
